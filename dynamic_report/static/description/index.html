Dynamic XLS REPORT

Get an Excel report of any Module.

Print the excel reports Directly from this module.

Its really easy. You can select any Model, can select fields those you want in the report, put domain on fields, put limit, order etc.

Now Save your time

You can get report of any model

How to use?

Just Install the module Dynamic Report Export

After Installing module click on the Dynamic Reports. A wizard will pop up.

Fields Description:
Model: Select the required Model Like Invoice to print the report of account.invoice.
Order BY: Select the field by which You want's to order the records.
Order: If you set it True than order will be descending according to the order by.
Limit: Enter the limit of records to print.
Value: If You set True in this field than it return the Name of many2one fields else-wise ID.
Select Fields: Select the fields those you want in report, Leave blank for all fields.
Domain Lines: Put the domain (search criteria) on fields.

Step 2

Fill the fields visible on the form. And select the fields in "Select Field" Tab. Leave blank if you want all fields to print.

Step 3

Select the values in "Domain Lines" tab to set the search criteria.

Step 4

Click the export button to get report and than click on the link to download the attachment.

Step 5

Excel report is ready.
